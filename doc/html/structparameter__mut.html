Detailed Description

Structure containing initialization parameters for mutation operator.

A structure that contains all the parameters for specifying a mutation operator. This data structure includes the mutation method, the mutation rate and variance (delta mutation), the lower and upper bounds for the fusion and non-uniform mutation operators. Furthermore, it contains the time and order parameters for the nonuniform mutation operator and a parameter that defines if the genome is of type int or real (float or double).
