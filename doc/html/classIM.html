Detailed Description

Island Model main class.

Island Model class contains methods to run an Island Model (IM) optimization. The class instance reads the topology from a file and builds a map (adjacency list). Moreover, provides methods for individuals selection and immigration over islands (threads).

Immigration method. It moves the immigrants (individuals) from one.

Moves immigrants from one island (thread) to another based on a predefined method.

Supported methods are:

  • random Individuals from the current island are randomly chosen and they are replaced by immigrants
  • elit The highest-fitness individuals within the current island population are replaced by immigrants
  • poor The lowest-fitness individuals within the current island population are replaced by immigrants
Parameters
num_immigrantsTotal number of immigrants the island can accept
unique_idIsland's unique ID (thread ID)
methodHow individuals from the local population will be replaced by immigrants
Returns
Nonthing (void)
Note
In a future version of GAIM, islands will support a variable number of immigrants.

size_t IM::read_connectivity_graph ( std::string  fname)

Island Model secondary operations methods.

Reads the connectivity graph (topology) from a text file.

Reads the connectivity graph (topology) from a file

The graph defines the number of islands (run on separate threads) and how they are connected. The format of the file is as follows: XX (number of islands - vertices) DST #edges SRC1 SRC2 SRC3

Example (ring topology with four islands): 4 0 1 3 1 1 0 2 1 1 3 1 2

Parameters
[in]fnameThe name of the file that contains the graph
Returns
num_vertices The number of vertices (islands) found in the file.
Note
The number of vertices is returned in order to validate the number of islands. In the rest of the code if the number if islands provided by the user does not match the number of vertices then the latter is being used as number of islands (threads).
