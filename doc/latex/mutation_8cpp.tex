 current genome) usable by default within G\+A\+IM. Some operators should be called multiple times per generation.

Extra Mutation functions.

Specifically, the mutation delta is drawn from the Normal/\+Gaussian distribution with mean 0 and standard deviation defined by the argument variance.

Non-\/uniform mutation operator. 

Performs a non-\/uniform mutation which is time-\/dependent, decaying over time. The mutation is computed as $ \Delta(b - x) $ or $\Delta(x - a) $ based on a random choice between -\/1 or 1, respectively.
